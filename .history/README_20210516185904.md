## 本地node+express部署打开步骤
安装express
```
cnpm install express -g //express框架
cnpm install express-generator -g //express项目生成插件
``` 
生成项目并启动
```
//新建文件夹，在该文件下依次执行如下命令
express myStudy //创建项目myStudy
cd myStudy //进入该目录
npm install //安装组件 
npm start //启动项目
```
打开
